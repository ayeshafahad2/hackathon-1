## 🎨 Design Tokens

### Colors
```css
/* Primary Colors */
--ifm-color-primary: #8b5cf6        /* Vibrant purple */
--ifm-color-primary-dark: #7c3aed   /* Darker purple */
--ifm-color-accent: #06b6d4         /* Cyan accent */

/* Dark Mode Grays */
--ifm-color-gray-0: #0a0a0f         /* Deepest background */
--ifm-color-gray-100: #12121a       /* Surface */
--ifm-color-gray-900: #ffffff       /* White */

/* Light Mode Grays */
--ifm-light-color-gray-0: #ffffff
--ifm-light-color-gray-900: #0f172a
```

### Typography
```css
--ifm-font-family-base: 'Plus Jakarta Sans'
--ifm-font-size-base: 16px
--ifm-line-height-base: 1.7
```

### Shadows
```css
--ifm-global-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
--ifm-global-shadow-lg: 0 20px 25px -5px rgba(0, 0, 0, 0.1)
--ifm-global-shadow-xl: 0 25px 50px -12px rgba(0, 0, 0, 0.25)
```

### Border Radius
```css
--ifm-global-radius: 12px
--ifm-global-radius-lg: 16px
--ifm-global-radius-xl: 24px
```

### Transitions
```css
--ifm-transition-fast: 150ms
--ifm-transition-normal: 250ms
--ifm-transition-slow: 400ms
```

## 💡 Tips for Customization

### Change Primary Color
Edit `frontend/src/css/custom.css`:
```css
--ifm-color-primary: #YOUR_COLOR;
```

### Adjust Animation Speed
```css
--ifm-transition-normal: 300ms; /* Slower */
--ifm-transition-normal: 200ms; /* Faster */
```

### Modify Chat Position
In `frontend/src/components/Root.tsx`:
```tsx
<ChatWidget position="bottom-left" />  // or "bottom-right"
```
