## 💡 Customization Tips

### Change Primary Color
Edit `frontend/src/css/custom.css`:
```css
--ifm-color-primary: #YOUR_COLOR;
```

### Adjust Animation Speed
```css
--ifm-transition-normal: 400ms; /* Slower */
--ifm-transition-normal: 200ms; /* Faster */
```

### Change Default Theme
Edit `frontend/docusaurus.config.ts`:
```ts
colorMode: {
  defaultMode: 'light',  // or 'dark'
}
```

### Modify Font
Edit `frontend/src/css/custom.css` import:
```css
@import url('https://fonts.googleapis.com/css2?family=YOUR_FONT');
--ifm-font-family-base: 'YOUR_FONT';
```
